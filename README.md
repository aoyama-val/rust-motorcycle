# Motorcycle game

[![Youtube Video](https://img.youtube.com/vi/NCZXKenqU-Y/0.jpg)](https://youtu.be/NCZXKenqU-Y)

## Requirement

- SDL 2
- Rust

## Controls

```
Up         : Speed up
Down       : Speed dorn
Left/Right : Rotate
Space      : Restart game
```
